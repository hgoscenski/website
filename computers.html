About My Computers

I typically name my desktop computers by their case i.e. the computer called H440 is built in an NZXT H440 case.
I have been building computers about once a year for the past couple of years and I am starting to have quite a collection.
The computers are basically for my own use in different capacities, some are used for different experiments, see H440v1, and some end up getting sold to fund the next one, see again H440v1.

SPEC-03

I built this computer for my family to replace an old computer using an AMD Athlon 64 x2.
That computer served us well but it was time for an upgrade and I spearheaded the operation.
This was the first time I built a computer from parts and it kicked off my enthusiasm to do so.

Refine

This was the first computer I built solely for myself. This computer is running Windows 10 Education Edition.
It, like all of my current computers, is running an Intel Processor, in this case an i7 4790K.
It is serving as a server for some of my utilities as well as serving as a virtualization host using Hyper-V.
Hyper-V is currently hosting an Ubuntu 16.10 Guest to use as a webserver as well as a remote shell.

H440v1

I built this one using an i7 6700K processor and then promptly used that base to construct a Hackintosh.
This was my first attempt at a Hackintosh and it went moderately well. It was 98% complete with only an issue with sleep.
I used it as a Hackintosh for a few months then sold it to fund the construction of H440v2.

H440v2

This is my most recent computer designed to be more powerful than anything else I have built.
It is running an i7 6850 hexacore processor, and its grotesquely underused for the power it has.
It serves as my gaming computer as well as a Windows environment to remote into when such a thing is called for.
This computer also serves as my daily use computer for all standard desktop usage.

Sony Vaio 2012

This is a laptop I picked up for a song, it was pretty high end back in 2012 and holds its own now.
It has both discrete and integrated graphics but since I don't use it for anything intense the discrete adapter is turned off.
I have run Arch Linux, Ubuntu, and Windows on this comptuer but right now it is running Windows 10 Pro.
This computer ends up bearing the brunt of experimentation when one of my desktops are not available.

Macbook Pro (Late 2015)

This is my daily driver machine, I am very much a fan of macOS, it manages to do everything I need it to do.
I use Homebrew as the package manager on this computer and I have it configured to act as my primary environment.
I have done various silly things to this computer, at one point installing Linux on it, however,
I have decided that macOS serves my needs well enough.
